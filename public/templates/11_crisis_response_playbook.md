## CRISIS #1: CASH RUNWAY < 8 WEEKS

### Immediate Actions (Day 1)
**Hour 1-2: Assessment**
- [ ] Calculate exact days of cash remaining
- [ ] Identify all pending receivables (with collection dates)
- [ ] List all committed payables (with due dates)
- [ ] Model daily burn rate with zero new revenue

**Hour 3-4: Emergency Meeting**
- [ ] COO + CFO + CEO emergency call
- [ ] Review all options (bridge loan acceleration, emergency fundraising, layoffs, shutdown)
- [ ] Decide on survival strategy

**Hour 5-8: Execute Plan A**
- [ ] Accelerate all receivables (call every distributor owing money)
- [ ] Defer all non-critical payables (negotiate with suppliers)
- [ ] Freeze all discretionary spending
- [ ] Stop all non-essential production

### Days 2-7: Stabilization
**Cash Generation:**
- [ ] Emergency sale of excess inventory (even at discount)
- [ ] Halt all marketing spend
- [ ] Reduce headcount to critical-only (painful but necessary)
- [ ] Negotiate payment plans with suppliers

**Emergency Fundraising:**
- [ ] Activate all investor relationships (founder network, angels, board)
- [ ] Offer aggressive terms (high interest, warrants, whatever it takes)
- [ ] Consider strategic acquirer discussions (if no other option)

## CRISIS #2: CO-PACKER FAILURE / PRODUCTION SHUTDOWN

### Immediate Actions (Hour 1)
- [ ] Assess current inventory (how many weeks of supply?)
- [ ] Calculate date when out-of-stock will occur
- [ ] Contact backup co-packer (emergency production request)
- [ ] Notify CEO and key distributors (transparency critical)

### Hour 2-4: Backup Production
**If Backup Co-Packer Available:**
- [ ] Emergency production run (even if higher cost)
- [ ] Expedite raw materials delivery
- [ ] Quality control shortcuts (if absolutely necessary)
- [ ] Premium for rush production (whatever it takes)

**If No Backup Available:**
- [ ] Call ALL co-packers in region (not just alcohol)
- [ ] Offer premium pricing (20-30% above market)
- [ ] Consider out-of-state co-packers (higher freight cost)
- [ ] Look for idle production capacity

### Days 2-7: Bridge the Gap
- [ ] Ration existing inventory to highest-velocity accounts
- [ ] Prioritize top 20% of distributors (80/20 rule)
- [ ] Communicate transparently with all customers
- [ ] Set up temporary production arrangement

### Long-Term Fix (Weeks 2-4)
- [ ] Secure permanent backup co-packer relationship
- [ ] Dual-source all critical production
- [ ] Build safety stock (4-6 weeks minimum)
- [ ] Never be dependent on single production source again

## CRISIS #3: MAJOR QUALITY ISSUE / RECALL

### Immediate Actions (Hour 1)
**Assess Scope:**
- [ ] How many units affected? (specific batch or all production?)
- [ ] What is the issue? (safety hazard, ABV off-spec, contamination, labeling error?)
- [ ] Is it a TTB compliance issue or quality issue?
- [ ] Risk to consumer safety? (THIS IS PRIORITY #1)

### Hour 2: Containment
- [ ] STOP all shipments immediately
- [ ] Quarantine affected inventory
- [ ] Trace distribution of affected batches
- [ ] Notify co-packer to stop production

### Hour 3-4: Regulatory Notification
**If Safety Issue:**
- [ ] Notify TTB within 24 hours (legal requirement)
- [ ] Consult legal counsel immediately
- [ ] Prepare recall plan (voluntary vs mandatory)
- [ ] Notify insurance carrier

**If Non-Safety Quality Issue:**
- [ ] Assess whether to notify TTB (consult legal)
- [ ] Decide on voluntary corrective action
- [ ] Plan customer communication

### Days 2-7: Recall Execution (If Required)
- [ ] Issue press release (if consumer safety issue)
- [ ] Notify all distributors with retrieval instructions
- [ ] Set up consumer hotline/email
- [ ] Track recall effectiveness (% of product retrieved)
- [ ] Root cause analysis (prevent recurrence)

### Cost Mitigation
- [ ] Insurance claim filing
- [ ] Co-packer liability discussion
- [ ] Supplier liability (if ingredient issue)
- [ ] Negotiate with distributors (return costs)

### Reputation Management
- [ ] Transparent communication (honesty builds trust)
- [ ] Social media response plan
- [ ] Customer compensation program (if appropriate)
- [ ] Independent quality audit (prove systems fixed)

## CRISIS #4: MAJOR CUSTOMER/DISTRIBUTOR LOSS

### Immediate Actions (Day 1)
**Assess Impact:**
- [ ] Lost revenue: $___/month
- [ ] Lost distribution points: ___
- [ ] % of total business: ___%
- [ ] Reason for loss: [Understand why]

**Hour 1-4: Damage Control**
- [ ] CEO call with lost customer (attempt to salvage)
- [ ] Understand root cause (price, quality, service, competition?)
- [ ] Document lessons learned
- [ ] Check contracts (any recourse?)

### Days 2-7: Backfill Strategy
**Replace Lost Volume:**
- [ ] Activate top 3 backup distributor prospects
- [ ] Offer aggressive terms to new distributors
- [ ] Reallocate inventory to remaining distributors
- [ ] Accelerate new door openings in other territories

**Financial Adjustment:**
- [ ] Revise revenue projections
- [ ] Cut costs proportionally (if significant loss)
- [ ] Adjust production schedule
- [ ] Update cash flow model

### Weeks 2-4: Prevent Future Losses
- [ ] Distributor health scorecard (early warning system)
- [ ] Quarterly business reviews with top 10 distributors
- [ ] Contract improvements (notice periods, termination clauses)
- [ ] Diversification strategy (no customer >20% of revenue)

## CRISIS #5: ALUMINUM CAN SHORTAGE

### Immediate Actions (Day 1)
**Inventory Assessment:**
- [ ] Current can inventory: ___ (__ weeks supply)
- [ ] Cans on order: ___ (ETA: ___)
- [ ] Production schedule at risk: [Dates]

### Hour 1-8: Alternative Sourcing
**Emergency Can Procurement:**
- [ ] Call ALL can suppliers (not just usual ones)
- [ ] Check can broker/distributor inventory
- [ ] Consider alternative can sizes (16oz if 12oz unavailable)
- [ ] Geographic arbitrage (buy cans from other regions)
- [ ] Accept higher pricing (survival > margin optimization)

**Alternative Packaging:**
- [ ] Brite cans + shrink sleeves (faster, lower MOQ)
- [ ] Digital can printing (if available)
- [ ] Bottles (if absolutely necessary - different format)
- [ ] Pouches (emerging format - explore)

### Days 2-7: Production Adjustment
- [ ] Prioritize highest-margin SKUs
- [ ] Reduce SKU count (focus on top 2 sellers)
- [ ] Batch size optimization (fewer runs, larger batches)
- [ ] Delay new product launches

### Long-Term Fix (Weeks 2-8)
- [ ] Secure long-term can supply agreement
- [ ] Dual-source can suppliers (never single-source again)
- [ ] Build can safety stock (8-12 weeks ideal)
- [ ] Explore captive can printing (if volume justifies)

## CRISIS #6: KEY EMPLOYEE DEPARTURE

### Immediate Actions (Day 1)
**Assess Impact:**
- [ ] Role departing: [Title]
- [ ] Reason for departure: [Understand why]
- [ ] Notice period: [How much time?]
- [ ] Critical knowledge at risk: [What do they know?]

**Hour 1-4: Retention Attempt**
- [ ] CEO + COO meeting with employee
- [ ] Understand root cause (money, role, culture, opportunity?)
- [ ] Counteroffer consideration (if worth retaining)
- [ ] Decision: Fight to keep or graceful exit?

### Days 2-7: Knowledge Transfer
**If Departure Confirmed:**
- [ ] Documented knowledge transfer (written SOPs)
- [ ] Cross-training other team members
- [ ] Extract all critical information (passwords, contacts, processes)
- [ ] Vendor/customer relationship handoffs

**Interim Coverage:**
- [ ] COO covers role temporarily (if critical)
- [ ] Fractional/consultant backfill (quick solution)
- [ ] Team redistribution of responsibilities
- [ ] Accelerate replacement hire

### Weeks 2-4: Replacement Search
- [ ] Job description refined (learn from departure)
- [ ] Aggressive recruiting timeline (2-4 weeks)
- [ ] Consider promoting from within
- [ ] Fractional executive as bridge (if specialized role)

### Prevent Future Departures
- [ ] Exit interview (brutal honesty)
- [ ] Team morale check (is this systemic?)
- [ ] Compensation benchmarking
- [ ] Retention plan for remaining key employees
